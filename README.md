# IBM Data Science Capstone

## SpaceX Falcon 9 Landing Prediction

This repository contains the completed IBM Data Science Professional Certificate Capstone Project.

### Repository Contents
- Jupyter notebooks (.ipynb)
- Python scripts (.py)
- Datasets (.csv)
- Presentation (.pdf/.pptx)

### Project Workflow
1. Data Collection using SpaceX REST API
2. Web Scraping using BeautifulSoup
3. Data Wrangling and Cleaning
4. Exploratory Data Analysis (EDA)
5. SQL Analysis
6. Interactive Visual Analytics (Folium & Plotly Dash)
7. Predictive Analysis using Machine Learning
8. Results and Conclusion

### Machine Learning Models
- Logistic Regression
- Support Vector Machine (SVM)
- Decision Tree
- K-Nearest Neighbors (KNN)
